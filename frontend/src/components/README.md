# Components

Reusable React UI components for the poker client.

## Components to Build

| Component | Purpose |
|-----------|---------|
| `Table/` | Renders the poker table: community cards, pot, player positions, dealer button |
| `Hand/` | Displays a player's hole cards (face up for self, face down for opponents) |
| `BettingControls/` | Fold, Check, Call, Raise buttons with amount slider for raises |
| `SpectatorPanel/` | UI for spectators to view odds and place wagers |
| `GameLog/` | Scrollable feed of game actions (bets, folds, reveals) |
| `WalletConnect/` | Connect/disconnect wallet button with address display |
